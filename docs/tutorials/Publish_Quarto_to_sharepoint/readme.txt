This is a tutorial on hos to publish a Quarto presentation using Reveal JS to Sharepoint

It contains to main files:

1. Quarto document that outputs an .html file with a detailed tutorial on how to publish quarto presentation to Sharepoint

- Publish_Quarto_presentation_in_sharepoint.qmd
- Publish_Quarto_presentation_in_sharepoint.html

2. A quarto presentation that has been turned into a .aspx file to be accessed as an interactive presentation from Sharepoint

The output of the .qmd presentation is the .html file with the same file name.
- ggplot2 themes 12 May 2023.qmd
- ggplot2 themes 12 May 2023.html
